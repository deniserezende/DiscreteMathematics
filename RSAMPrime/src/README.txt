Para executar a main, utilize a seguinte linha de comando:
python3 main.py -a {numero de primos} -s {tamanho máximo em bits do produto dos primos} -{x}
onde {x} é opcional
x = b : para início da coleta de dados
x = c : para continuar a coleta de dados
x = e : para finalizar a coleta de dados

Note que: será produzido como output no máximo dois arquivos 'privatekeysdfr.txt', esse contém as chaves privadas da última execução e'RSADeniseReport.csv', caso for pedida a coleta de dados. 

Para visualizar uma tabela com os resultados de 'RSADeniseReport.csv', basta seguir as instruções contidas no arquivo report_data.py
